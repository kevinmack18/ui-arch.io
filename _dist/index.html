Button

Buttons can be used with <a>, <button>, and <input> elements.

Types of buttons:

  • Default: Standard button
  • Primary: Provides extra visual weight and identifies the primary action in a set of buttons
  • Success: Indicates a successful or positive action
Example 
<button class="btn btn-default">Default</button>
<button class="btn btn-primary">Primary</button>
<button class="btn btn-success">Success</button>

Label

Label can be used with <label> elements.

Types of Loaders:

  • t1: Standard button
  • t2: Provides extra visual weight and identifies the primary action in a set of buttons
Example 
    <label class="c-label">I'm a label</label>

Search Input

Loaders can be used with <span>, <div>, and any "block type" elements.

Types of Loaders:

  • Default: Standard button
  • Primary: Provides extra visual weight and identifies the primary action in a set of buttons
  • Success: Indicates a successful or positive action
Example 
    <input class="c-input-search" />
